# Evoluzione-del-Potere-d-Acquisto-in-Italia-2016-2024
## Descrizione dell'analisi:
Questo repository contiene i dati utilizzati per l'analisi dell'evoluzione del potere d'acquisto in Italia tra il 2016 e il 2024. Lo studio esamina le relazioni tra reddito, inflazione, disoccupazione, povertà e disuguaglianza concentrandosi sui divari territoriali tra macroaree (Nord-Ovest, Nord-Est, Centro e Mezzogiorno).

## Contenuti del Repository:
- Report dell'analisi in PDF.
- Dati utilizzati per l'analisi sono organizzati in un file Excel (dati.xlsx) con diversi fogli, ciascuno contenente una specifica tipologia di informazione.
- File shapefile con i confini delle regioni italiane (fonte ISTAT), utilizzato per la creazione delle mappe del potere d'acquisto.

## Strumenti e Software Utilizzati
- Python per la pulizia dei dati grezzi provenienti da ISTAT ed EUROSTAT.
- R per l'analisi statistica dei dati, la cluster analysis, la PCA e la creazione di grafici e tabelle.
- QGIS per la realizzazione delle mappe geografiche del potere d'acquisto regionale.
