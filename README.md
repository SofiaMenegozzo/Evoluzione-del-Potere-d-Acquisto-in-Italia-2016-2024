# Evoluzione-del-Potere-d-Acquisto-in-Italia-2016-2024
Questo repository contiene i dati utilizzati per l'analisi dell'evoluzione del potere d'acquisto in Italia tra il 2016 e il 2024. Lo studio esamina le relazioni tra reddito, inflazione, disoccupazione, povertà e disuguaglianza concentrandosi sui divari territoriali tra macroaree (Nord-Ovest, Nord-Est, Centro e Mezzogiorno).
